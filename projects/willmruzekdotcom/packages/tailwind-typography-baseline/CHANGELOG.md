# Changelog

All notable changes to the Tailwind CSS Typography Baseline Plugin will be documented in this file.

The format is based on [Conventional Commits](https://www.conventionalcommits.org/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changes

- **style**: remove bottom margin from list items (`li`) in baseline typography styles

## [0.1.0] - 2025-08-27

### Features

- **feat**: add complete baseline grid system with 24px vertical rhythm
- **feat**: add dynamic prefix support using Tailwind CSS v4 `@utility` directives
- **feat**: add type scale system with 5 sizes (sm, base, lg, xl, 2xl)
- **feat**: add responsive design support with breakpoint modifiers
- **feat**: add development overlay utility (`prose-baseline-grid`) for visual debugging
- **feat**: add comprehensive typography elements support (headings, paragraphs, lists, tables, code, images, media)
- **feat**: add CSS architecture with 2 core variables and `calc()` expressions
- **feat**: add CSS custom properties with `--tw-baseline-` namespace for conflict prevention
- **feat**: add proper `not-prose` class exclusion support
- **docs**: add comprehensive documentation with usage examples and setup instructions
