Perfect vertical rhythm for your content

Until now, trying to maintain consistent vertical rhythm in your typography has been a tedious task that required careful calculations and a lot of complex custom CSS.

By default, Tailwind provides excellent typography utilities, but maintaining a consistent baseline grid requires additional work. The prose-baseline plugin ensures every element aligns to a configurable baseline grid (24px by default), creating perfect vertical rhythm throughout your content.

Note: This demo uses responsive baseline scaling (24px → 28px → 32px at larger breakpoints) to showcase how the plugin adapts to different screen sizes. This responsive behavior is implemented in the demo's CSS and shows what's possible when you customize the baseline unit.

This plugin is designed to work seamlessly with Tailwind CSS v4, providing:

  • Configurable baseline grid alignment for all typography elements
  • Responsive type scales matching Tailwind's size conventions
  • Visual debugging tools to verify alignment

The prose-baseline class can be applied to any block of HTML content to enforce consistent vertical rhythm: 

<article class="prose-baseline">
  <h1>Your Title Here</h1>
  <p>
    All text will align to a perfect baseline grid,
    ensuring consistent spacing and visual rhythm.
  </p>
  <!-- ... -->
</article>

For more information about how to use the plugin and the features it includes, read the documentation.

Code should align to the baseline too

Code blocks are particularly tricky when it comes to baseline grids. The prose-baseline plugin ensures that both inline code snippets and code blocks maintain proper alignment.

Here's what a typical CSS import looks like:

@import "tailwindcss";
@import "tailwind-typography-baseline/baseline-grid.css";

@layer base {
  :root {
    --tw-baseline-unit: 24;
  }
}

And here's how you'd use it in your HTML:

<article class="prose-baseline prose-baseline-lg">
  <h1>Perfect Vertical Rhythm</h1>
  <p>Every element aligns to the baseline grid.</p>
</article>

What is a baseline grid?
A baseline grid is an invisible structure used in typography to align text elements. It ensures consistent vertical spacing and creates visual rhythm in layouts.
Why is vertical rhythm important?
Vertical rhythm creates a sense of visual harmony and makes content easier to read. When text aligns to a consistent grid, the reader's eye can move smoothly through the content.
How does this plugin work?
The plugin uses CSS custom properties and careful calculations to ensure every typographic element—from headings to paragraphs to list items—aligns to a configurable baseline grid. It's all done with pure CSS, no JavaScript required.
